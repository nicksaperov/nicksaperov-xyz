# BTRFLY — agent instructions

React + Vite + Tailwind 4. Entry: `src/App.jsx`.

## Structure

- Put reusable UI in `src/components/` (one component per file, PascalCase).
- Import components into `App.jsx` or parent components; keep `App.jsx` as layout/composition only when splitting.
- Use existing Tailwind patterns from `App.jsx` (spacing, typography, grayscale images).

## Commands

- `npm install` — first-time setup
- `npm run dev` — local dev server
- `npm run build` — production build
